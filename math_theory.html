5. Important Algebra Formulas

  • (a + b)² = a² + b² + 2ab
  • (a - b)² = a² + b² - 2ab
  • (a + b)² + (a - b)² = 2(a² + b²)
  • (a + b)² - (a - b)² = 4ab
  • a² - b² = (a + b)(a - b)
  • (a + b)³ = a³ + b³ + 3ab(a + b)
  • (a - b)³ = a³ - b³ - 3ab(a - b)
  • a³ + b³ = (a + b)(a² - ab + b²)
  • a³ - b³ = (a - b)(a² + ab + b²)
  • (a + b + c)² = a² + b² + c² + 2(ab + bc + ca)
